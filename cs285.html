STTY

the stty command in unix prints and changes characteristics of the terminal.

The idea: The tty driver controls input and output to the terminal. We can read and change the driver settings. For example, remap the backspace function to a different key, turn off echo to hide typed text, etc.

The struct termios can hold the information about the terminal settings.

function tcgetattr(int, const struct termios) will fill the struct with the current settings.
Then, we can display the settings.

function tcsetattr(int file_descriptor, int when, const struct termios) will change the settings.
By passing a modified struct back, we can make the desired changes.

link to manpage: https://www.man7.org/linux/man-pages/man1/stty.1.html
